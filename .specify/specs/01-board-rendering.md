# Spec 01: 체스 보드 렌더링

**Status**: Implemented
**Created**: 2026-03-18 (역추적)

---

## Overview

터미널에서 체스 보드를 시각적으로 표시하는 기본 기능.

---

## User Stories

### US1: 표준 초기 배치 보기

**Given** 사용자가 체스 에디터를 실행하면
**When** 프로그램이 시작되면
**Then** 표준 체스 초기 배치가 터미널에 표시된다
**And** 체스판 좌표(파일 a-h, 랭크 1-8)가 함께 표시된다

### US2: 체스말 구분

**Given** 체스 보드가 표시될 때
**When** 사용자가 보드를 보면
**Then** 백/흑 각 색깔의 모든 기물이 유니코드 심볼(♔♕♖♗♘♙ / ♚♛♜♝♞♟)로 구분되어 보인다
**And** 밝은/어두운 칸이 체커보드 패턴으로 교차 표시된다

---

## Functional Requirements

- **FR-001**: 8×8 체스 보드를 터미널에 렌더링한다
- **FR-002**: 체스말은 유니코드 심볼로 표시한다
- **FR-003**: 칸은 밝은/어두운 배경색으로 체커보드 패턴을 형성한다
- **FR-004**: 파일(a-h) 및 랭크(1-8) 좌표를 표시한다
- **FR-005**: 표준 초기 배치(standard position)를 제공한다
- **FR-006**: Rich 라이브러리를 사용해 TUI로 표시한다

---

## Success Criteria

- [x] `uv run chess-editor` 실행 시 오류 없이 보드가 출력된다
- [x] 백/흑 기물 12종 모두 올바른 유니코드 심볼로 표시된다
- [x] 체커보드 패턴이 올바르게 렌더링된다
- [x] 파일/랭크 레이블이 표시된다

---

## Out of Scope

- 인터랙티브 편집 (클릭/키보드 입력)
- FEN 불러오기/저장
- 기물 이동 검증
