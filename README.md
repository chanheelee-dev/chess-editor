# Chess Editor

체스 보드를 커스텀하게 만들 수 있는 터미널 기반 체스 에디터입니다.

## 기능

- 🎨 Rich 라이브러리를 사용한 아름다운 터미널 UI
- ♟️ 유니코드 체스 기물 표시
- 📦 표준 체스 시작 포지션 지원
- 🔧 커스텀 보드 포지션 생성 (향후 지원 예정)

## 설치

이 프로젝트는 `uv`를 사용하여 관리됩니다.

```bash
# 의존성 설치
uv sync

# 개발 모드로 실행
uv run chess-editor
```

## 사용법

```bash
# 체스 보드 에디터 실행
uv run chess-editor
```

현재는 표준 체스 시작 포지션을 보여주는 데모 버전입니다.

## 프로젝트 구조

```
chess-editor/
├── src/
│   └── chess_editor/
│       ├── __init__.py
│       ├── main.py           # 메인 진입점
│       ├── board.py          # 체스 보드 모델
```
